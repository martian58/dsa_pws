# Practical Work 9

This project contains codes for DSA Practical Work 9. (Press Ctrl + Shift + V)


Compile and test on `debian` based systems(ubuntu, kali).


## Project Structure
- `include/`: Header files.
- `src/`: Source code.
- `test/`: Test files.
- `Makefile`: Compilation automation.

## Build and Run
1. Build the project:

   ```bash
    make
    ```

2. Run the project:

    ```bash
    ./main
    ```
    For exercise 4:

    ```bash
    ./ex4
    ```
    
3. Clean the bins:

    ```bash
    make clean
    ```
