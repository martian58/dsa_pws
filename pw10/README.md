# Practical Work 10, 11

This project contains codes for DSA Practical Work 10,11. (Press Ctrl + Shift + V)


Compile and test on `debian` based systems(ubuntu, kali).


## Build and Run
1. Build the project:

   ```bash
    make
    ```
    Or:

    ```bash
    make run
    ```

2. Run the project:

    ```bash
    ./election
    ```
    
3. Clean the bins:

    ```bash
    make clean
    ```
